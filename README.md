## کار باقی‌مانده قبل از build

⚠️ در فایل `www/js/auth.js`، مقدار `CLIENT_ID` را با Client ID که از Google Cloud
Console گرفته‌اید جایگزین کنید.

## چطور APK بسازم؟

هر بار که تغییری را به شاخه `main` push کنید، یک GitHub Action به صورت خودکار
اجرا می‌شود و فایل APK می‌سازد. برای دریافت فایل:

1. به تب **Actions** در مخزن گیت‌هاب بروید
2. روی آخرین workflow کامل‌شده کلیک کنید
3. در بخش **Artifacts** فایل `app-debug-apk` را دانلود کنید
4. فایل zip را باز کنید و APK داخلش را روی گوشی نصب کنید

(ممکن است لازم باشد نصب از منابع ناشناس را در تنظیمات اندروید فعال کنید)
